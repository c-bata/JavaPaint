JavaPaint
=========

Java+Swingでお絵かきソフト

まだまだ開発中...

### 追加機能一覧

- メニューバーの作成
	- 「終了ボタン」
	- 「新規ファイル作成」
	- 「元に戻す」の実装
	- 「やり直す」の実装
	- 「グリッド線」の表示
	- 「画像形式の選択」
	- メニューアイテムにキーボードニーモニックの設定
	- メニューアイテムにキーボードアクセラレーターの設定
- ファイル選択ダイアログの実装
- ファイルチューザーダイアログのフィルタ設定
- グリッド線に沿った補正機能
- スライダーによる線の太さの指定
	- スライダーには,目盛りをつけている.
